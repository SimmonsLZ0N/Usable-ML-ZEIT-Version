What is Machine Learning?

Machine Learning, is a field of artificial intelligence that enables prediction of events as well as recognition of patterns by abstracting from a given dataset. This application covers a typical machine learning task: recognition of hand written digits.

Understanding CNNs

The animation shows how models are able to recognize hand written digits from MNIST - a database containing 60000 handwritten digits.

The moving red box in the animation is called a kernel or filter, and it slides over the image to detect features like edges and patterns. The result of this process is what we call a feature map, shown on the right, which highlights important details the network will use to make decisions.
